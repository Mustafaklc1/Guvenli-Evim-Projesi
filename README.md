# Güvenli Evim - Deprem Farkındalık Simülasyonu 🌍

Bu proje, deprem anında yapılması gereken doğru davranışları öğretmeyi amaçlayan eğitici bir simülasyon oyunudur. Unity oyun motoru ile fizik tabanlı etkileşimler kullanılarak geliştirilmiştir.

## 🎮 Oyunun İçeriği
Oyunda bir aile karakterleri üzerinden, deprem esnasında alınması gereken hayati önlemler ve doğru hareket tarzları interaktif bir şekilde gösterilmektedir. Temel amaç, oyunculara kriz anında ne yapmaları gerektiğini görsel ve uygulamalı olarak aktarmaktır.

## 🚀 Nasıl Oynanır?
1. Sayfanın sağ tarafındaki **Releases** bölümünden oyunun son sürümünü (ZIP dosyası) indirin.
2. İndirdiğiniz dosyayı sağ tıklayıp klasöre çıkartın.
3. Klasörün içindeki `.exe` (Uygulama) dosyasına tıklayarak simülasyonu başlatın.
